# คลื่นเพลง · Music Wave Player

แอปเล่นเพลง MP3 MP4 / วิดีโอ แบบไฟล์เดียว (HTML+CSS+JS ล้วน ไม่ต้องติดตั้งอะไรเพิ่ม)

## ฟีเจอร์
- 🎵 เล่นไฟล์ MP3 / 🎬 เล่นไฟล์ MP4 — เลือกได้ **ทีละหลายไฟล์พร้อมกัน** ผ่านแท็บ "เพลย์ลิสต์"
- 🗂 **เพลย์ลิสต์ที่บันทึกไว้ในเครื่อง** — เพลง/วิดีโอที่นำเข้าจะถูกเก็บไว้ในเบราว์เซอร์ (IndexedDB + localStorage) เปิดหน้าเว็บใหม่ก็ยังอยู่, เล่นจบเพลงหนึ่งจะเล่นเพลงถัดไปในเพลย์ลิสต์ให้อัตโนมัติ, กด ✕ เพื่อลบออกจากรายการ
- 🖼 **แสดงปกอัลบั้มจริง** — ดึงรูปปกที่ฝังอยู่ในไฟล์ MP3 (ID3 tag) มาแสดงบนแผ่นเสียงหมุนอัตโนมัติ ส่วนไฟล์ MP4 จะจับภาพจากวิดีโอมาทำเป็นปกให้ ถ้าไฟล์ไม่มีรูปปกฝังไว้ ระบบจะโชว์ไอคอนเพลงแทน
- 🌐 เล่นจาก URL โดยตรง (วางลิงก์ไฟล์ .mp3/.mp4)
- ❤️ บันทึกรายการโปรด (เก็บไว้ระหว่างที่เปิดหน้าเว็บอยู่)
- 🔍 ค้นหาสถานี/เพลง
- 🎼 ชื่อเพลง/ศิลปิน
- ⏩ ปรับความเร็วเล่น 0.5x–2x
- 📺 โหมดเต็มจอ
- 📱 ปรับขนาดอัตโนมัติสำหรับมือถือ / แท็บเล็ต / ทีวี (จอกว้าง)

## หมายเหตุ
- เพลย์ลิสต์ (ไฟล์เพลง + ปกอัลบั้ม) เก็บไว้ในเบราว์เซอร์ของอุปกรณ์นั้นๆ ด้วย IndexedDB/localStorage — ใช้งานได้เต็มรูปแบบเมื่อนำไฟล์ไปโฮสต์จริงบน GitHub Pages หรือเซิร์ฟเวอร์อื่น **แต่ถ้าเปิดดูตัวอย่างในหน้าต่างแชทนี้ การบันทึกอาจไม่ทำงาน** เนื่องจากหน้าต่างพรีวิวถูกจำกัดสิทธิ์การเข้าถึงพื้นที่จัดเก็บของเบราว์เซอร์
- รายการโปรดเก็บไว้ในหน่วยความจำของเบราว์เซอร์ระหว่างเปิดหน้าเว็บ (รีเฟรชหน้าแล้วจะรีเซ็ต)
- สถานีเพลงตัวอย่างในหน้าแรกเป็นการ์ดตัวอย่าง (ยังไม่มีไฟล์เสียงจริงผูกไว้) — กดแล้วให้ใส่ไฟล์หรือ URL จริงเพื่อเริ่มฟัง
- การดึงปกอัลบั้มจาก MP3 ใช้ไลบรารี jsmediatags ผ่าน CDN (ต้องมีอินเทอร์เน็ตตอนโหลดหน้าเว็บครั้งแรก)
